How to Run the Professional AI Storyboard Generator

🚀 Step 1: One-Time Setup

Complete this only the first time you use the skill.

Navigate to the skill directory:

cd /Users/abhinandan/plugins/ebc-storyboard/skills/storyboard

Copy the environment template:

cp examples/.env.example .env

Edit .env and add your API keys:

nano .env

Add these lines to the .env file:

MIDJOURNEY_API_KEY=afk_your_key_here
RUNWAY_ML_API_KEY=key_your_key_here
💡 Tip: Get your keys from:
  • Midjourney: https://apiframe.pro
  • Runway ML: https://app.runwayml.com

📝 Step 2: Generate a Storyboard

From the skill directory, run one of these commands:

Option A: Basic HTML only (no videos)

node storyboard-generator.js your-script.txt --output "Module Name"

Option B: With AI images + B-roll videos (RECOMMENDED)

node storyboard-generator.js your-script.txt --video --output "Module Name"

Option C: Without image generation (placeholders only)

node storyboard-generator.js your-script.txt --no-images --output "Module Name"

⚙️ Command Options Reference

node storyboard-generator.js <script-file> [options]

Options:
  --video              # Generate B-roll videos (Runway ML)
  --no-images          # Skip image generation (use SVG placeholders)
  --output NAME        # Output filename (default: storyboard)

Examples:

# Basic HTML
node storyboard-generator.js training.txt

# With images and videos
node storyboard-generator.js training.txt --video

# With custom output name
node storyboard-generator.js training.txt --video --output "My Module"

# With placeholders only
node storyboard-generator.js training.txt --no-images --output "Draft"

💡 Pro Tips

Tip 1: Start Simple

First time? Start without videos to test:

node storyboard-generator.js test-script.txt --output "test"

Quick (30 seconds), no API calls needed.

Tip 2: Full Production

Use videos for final, professional output:

node storyboard-generator.js script.txt --video --output "Final Module"

Complete (5-7 minutes), professional cinema-grade quality.

Tip 3: Team Sharing

Bundle everything for easy sharing:

zip -r "Module.zip" "Module.html" "Module_videos/"

Send the zip to your production team. Everything is self-contained.

Tip 4: Check Costs

View the cost breakdown:

cat "Module-metrics.json" | jq '.imageMetrics, .videoMetrics'

See exactly what you spent on images and videos.
