Storyboard Skill — Setup Guide

Turn a video script into a production-ready HTML storyboard in seconds. This guide gets you set up in under two minutes.

Requirements

Claude Code (desktop or CLI)
Git — pre-installed on macOS
No GitHub account needed
Don't have Claude Code? Download it at claude.ai/code — available as a Mac desktop app, Windows app, or CLI.

One-time installation

1

Clone the skill to your machine

Open Terminal and run:

git clone https://github.com/AbhiMalik/ebc-storyboard ~/plugins/ebc-storyboard
This downloads the skill files to a plugins folder in your home directory. You only need to do this once.
2

Create a shortcut command (optional but recommended)

Every time you want to use the skill, you need to launch Claude Code with a long command. You can create a short nickname for it so you don't have to type the full thing each time.

Run these two lines in Terminal:

echo 'alias storyboard="claude --plugin-dir ~/plugins/ebc-storyboard"' >> ~/.zshrc source ~/.zshrc
This registers storyboard as a shortcut. From now on, whenever you open Terminal and type storyboard and press Enter, it is exactly the same as typing the full claude --plugin-dir ~/plugins/ebc-storyboard command. You only need to do this setup once.

Using the skill

1

Launch Claude Code with the plugin loaded

Open Terminal and run the following command. This opens a Claude Code session that has the storyboard skill available.

claude --plugin-dir ~/plugins/ebc-storyboard

If you created the shortcut in the setup step, you can just type this instead:

storyboard
Note: You need to do this every time — the plugin is not permanently installed. Each Claude Code session needs to be started this way for the skill to be available.
2

Invoke the skill with your script

Type /ebc-storyboard:storyboard, then paste your video script:

/ebc-storyboard:storyboard [Paste your full video script here]

To specify a filename, add it on the first line:

/ebc-storyboard:storyboard Save as module-3-mens-rea.html [Your script]
3

Open the output file in your browser

Claude saves a self-contained .html file in your current folder. Double-click it to open — no internet connection or server needed.

Each slide shows the audio script on the left with inline ▶ cue markers marking exactly when each animation element should appear, and the visual (instructor shot or animatic) on the right.

Getting updates

1

Open Terminal and go to the plugin folder

cd ~/plugins/ebc-storyboard
2

Pull the latest files from GitHub

git pull
That's it. No reinstall or restart needed — the next time you launch the skill, Claude Code will automatically use the updated files from disk.
